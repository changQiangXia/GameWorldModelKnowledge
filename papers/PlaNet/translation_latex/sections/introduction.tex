\section{引言}
\label{introduction}

\afterpage{\input{figures/domains/figure}}

对于动力学已知的决策问题，规划是一种自然且有效的方法，典型例子包括博弈和模拟机器人控制\citep{tassa2012mpc,silver2017alphago,moravvcik2017deepstack}。在未知环境中进行规划时，智能体必须从经验中学习环境动力学。如何学到足以支撑规划的准确动力学模型，一直是一个难题。主要困难包括模型误差、多步预测中的误差累积、无法刻画多个可能未来，以及在训练分布之外给出过度自信的预测。

与无模型强化学习相比，使用学习到的模型进行规划有几个优势。首先，基于模型的规划能够利用更丰富的训练信号，不需要通过 Bellman 备份传播奖励，因此通常更具数据效率。其次，规划可以通过增加动作搜索的计算预算来提升性能，这一点已由\citet{silver2017alphago}展示。最后，学习到的动力学模型可以与具体任务解耦，因此有机会迁移到同一环境中的其他任务。

近期工作已经能在简单低维环境中学习动力学模型\citep{deisenroth2011pilco,gal2016deeppilco,amos2018awareness,chua2018pets,henaff2018planbybackprop}。不过，这些方法通常假设可以访问环境的底层状态和奖励函数，而实际应用中未必具备这样的条件。在高维环境中，我们希望在紧凑的潜在空间中学习动力学，从而支持快速规划。此前，潜在模型的成功主要局限于较简单的任务，例如从稠密奖励中学习平衡 cartpole 或控制双连杆机械臂\citep{watter2015e2c,banijamali2017rce}。

本文提出 \fullmethod（\method），一种基于模型的智能体。它从像素中学习环境动力学，并在紧凑的潜在空间中通过在线规划选择动作。为了学习动力学，我们使用同时包含随机成分和确定性成分的转移模型；此外，还实验了一种鼓励多步预测的广义变分目标。\method 能够从像素输入解决连续控制任务，这些任务比此前通过学习模型进行规划所解决的任务更具挑战性。

本文的主要贡献如下：
\begin{itemize}
\paritem{Planning in latent spaces}
我们通过学习动力学模型并在其潜在空间中高效规划，解决了\cref{fig:domains}所示的多种 DeepMind Control Suite 任务。最终性能上，我们的智能体显著优于无模型 A3C，并在部分任务上超过 D4PG；同时，平均只需要约 $1/200$ 的环境交互量，计算时间相近。
\paritem{Recurrent state space model}
我们设计了一种同时包含确定性成分和随机成分的潜在动力学模型\citep{buesing2018dssm,chung2015vrnn}。实验表明，这两个组成部分对取得较高规划性能都很关键。
\paritem{Latent overshooting}
我们将标准变分下界推广到多步预测。该目标只使用潜在空间中的项，因此可以作为高效正则项改进长期预测，并与任意潜在序列模型兼容。
\end{itemize}
