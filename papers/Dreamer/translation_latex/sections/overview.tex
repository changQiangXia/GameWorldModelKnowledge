\paragraph{强化学习}

我们将视觉控制表述为部分可观测马尔可夫决策过程（POMDP）。时间步为离散的 $t \in [1; T]$，动作是连续向量 $a_t\sim \p(a_t|o_{\leq t},a_{<t})$，观测为高维图像，并带有标量奖励 $o_t,r_t\sim\p(o_t,r_t|o_{<t},a_{<t})$。目标是学习一个智能体，使期望累计奖励 $\smash{\E[\big]{p}{\sum_{t=1}^T r_t}}$ 最大化。\Cref{fig:tasks} 展示了本文使用的任务。

\paragraph{智能体组件}

在想象中学习的智能体通常包含三个部分：动力学学习、行为学习和环境交互\citep{sutton1991dyna}。Dreamer 通过在世界模型的紧凑潜在空间中预测假想轨迹来学习行为。如 \cref{fig:schema} 和 \cref{alg:agent} 所示，在智能体生命周期内，Dreamer 反复执行以下操作，这些操作也可以交替或并行进行：
\begin{itemize}
\item 从过去经验构成的数据集中学习潜在动力学模型，用动作和过去观测预测未来奖励。我们在 \cref{sec:dynamics} 中讨论用于学习潜在动力学的现有方法。
\item 从预测出的潜在轨迹中学习动作模型和价值函数模型，如 \cref{sec:behavior} 所述。价值函数模型通过想象奖励优化 Bellman 一致性，动作模型则通过神经网络动力学反传价值估计的梯度来更新。
\item 在真实环境中执行学得的动作模型，收集新经验并扩充数据集。
\end{itemize}

\paragraph{潜在动力学}

Dreamer 使用由三个组件构成的潜在动力学模型。表征模型编码观测和动作，生成具有马尔可夫转移的连续向量模型状态 $s_t$\citep{watter2015e2c,zhang2018solar,hafner2018planet}。转移模型在看不到未来观测的情况下预测未来模型状态；奖励模型根据模型状态预测奖励。
\begin{gather}
\begin{aligned}
\makebox[12em][l]{表征模型：} && &\p(s_t|s_{t-1},a_{t-1},o_t) \\
\makebox[12em][l]{转移模型：} && &\q(s_t|s_{t-1},a_{t-1}) \\
\makebox[12em][l]{奖励模型：} && &\q(r_t|s_t).
\label{eq:latent_dynamics}
\end{aligned}
\end{gather}%
我们用 $p$ 表示在真实环境中产生样本的分布，用 $q$ 表示学得的模型。具体而言，转移模型让我们能够在紧凑潜在空间中向前预测，而不需要观测或想象对应图像。这降低了内存占用，也使快速预测数千条想象轨迹成为可能。

该模型类似于非线性 Kalman 滤波器\citep{kalman1960filter}、潜在状态空间模型，或带连续状态的 HMM。不过，它以动作为条件并预测奖励，因此智能体可以在不实际执行动作的情况下，想象候选动作序列可能带来的结果。
